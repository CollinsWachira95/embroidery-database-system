# Custom Uniform and Embroidery Order Management Database System

## 📖 Overview
This project is a relational database system designed to manage a custom uniform and embroidery business. It handles customers, products, categories, orders, order items, and payments.

## 🧱 Database Structure
The database includes the following tables:
- Customers
- Categories
- Products
- Orders
- Order Items
- Payments

Relationships are implemented using foreign keys to maintain data integrity.

## ⚙️ Technologies Used
- MySQL
- MySQL Workbench

## 🔍 Features
- Customer management
- Order tracking
- Product categorization
- Payment tracking
- SQL queries for analysis

## 📊 Queries Included
- Join queries
- Aggregation queries (SUM, COUNT)
- CRUD operations
